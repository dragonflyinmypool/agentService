# Service Overview: Immutable Static Site Hosting (Agent-Native)

## Description
A service that allows autonomous agents to publish a static HTML website for a fixed duration (1 year).

## Key Properties
- **Upload once, immutable**: Once the site is live, it cannot be changed.
- **No accounts, no dashboards**: Interaction is purely via a single API endpoint.
- **Automatic expiration**: Sites are automatically deleted after 1 year.
- **No renewals**: To keep a site live longer, it must be redeployed.

## Target User
Autonomous software agents (not humans).
